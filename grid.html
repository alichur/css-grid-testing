This grid uses grid-column and grid-row to specify the sizes of a box. E.g. the header is
grid-column: 1 / 4;
- a row that extends from column 1 to 4. The second box is
grid-row: 2 / 4;
a column that extends from row 2 to row 4.

This grid uses template areas to define the sections. Don't try this with IE! e.g.

grid-template-areas:
"header header header"
"nav main main"
"nav single single"
}

Each div has a class which defines which area it belongs to

.header-area {
grid-area: header;
}
In firefox dev tools select "display area names" to see more.
